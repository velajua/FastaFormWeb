# FastaFormWeb

Static GitHub Pages site for FastaForm.

Expected public URL after publishing the `velajua/FastaFormWeb` repository:

```text
https://velajua.github.io/FastaFormWeb/
```

## Local checks

```powershell
npm.cmd test
```

The site has no build step. Open `index.html` directly, or serve the folder with any static HTTP server.

## GitHub Pages setup

1. Create a public GitHub repository named `FastaFormWeb` under `velajua`.
2. Push this folder as the repository contents.
3. In GitHub, go to `Settings` -> `Pages`.
4. Set the source to deploy from the default branch root.
5. Use `https://velajua.github.io/FastaFormWeb/privacy/` as the Google Play privacy policy URL after the page is live.
